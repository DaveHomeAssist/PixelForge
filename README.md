# PixelForge

PixelForge is a lightweight browser-based drawing surface that mixes raster painting and vector shape layers in a single React app.

## Features

- Raster brush and eraser tools
- Vector rectangle, ellipse, and line tools
- Layer visibility, opacity, and blend modes
- Project save/load support
- PNG export
- Keyboard shortcuts for common drawing actions

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

GitHub Pages deployment is handled by `.github/workflows/deploy-pages.yml`.
